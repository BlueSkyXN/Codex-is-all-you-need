## Purpose

Plan a presentation before creating slides.

## Inputs

Use provided materials, such as:

- project notes
- reports
- research summaries
- meeting notes
- metrics
- user-provided audience and objective

## Process

1. Clarify audience, objective, decision needed, and time limit.
2. Define the core message in one sentence.
3. Build a storyline before page details.
4. Create a slide-by-slide outline.
5. For each slide, specify title, key message, supporting points, suggested visual, and speaker notes.
6. Identify missing data, weak evidence, or unclear audience needs.

## Recommended structure

```text
1. Audience and objective
2. One-sentence core message
3. Storyline
4. Slide outline
5. Speaker notes
6. Missing information
```

## Rules

- Do not invent data, dates, owners, decisions, or customer quotes.
- Do not overfill slides.
- Prefer decision-oriented slide titles over topic labels.
- Use evidence from materials for claims.

## Output

Return:

1. Audience and objective
2. Core storyline
3. Slide-by-slide outline
4. Speaker notes
5. Missing information
6. Suggested next actions
