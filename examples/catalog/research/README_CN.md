# Research Catalog / 研究目录

[English](README.md) | 中文

这里存放研究材料整理、证据映射、缺口识别、来源去重和综合写作相关 agents 与 skills。

## 内容清单

```text
agents/   4 个研究 agents
skills/   3 个公开 skills
```

## Agent 角色

- `research_material_processor`：在综合写作前盘点、分类、去重和结构化已收集研究材料。
- `research_evidence_mapper`：将 claims 映射到来源、章节、摘录和置信度。
- `research_gap_finder`：识别缺失证据、冲突、弱假设、无依据结论和后续问题。
- `research_synthesis_writer`：基于已映射证据起草 brief、综合笔记、对比摘要和报告。

## Skills

- `source-dedup`：去重重叠的研究材料、AI 笔记、摘录、转写和 source packs。
- `evidence-table`：基于给定材料建立 claim/evidence/source 表。
- `research-synthesis`：将已映射证据整理为平衡的研究综合，并说明限制。

## 维护说明

不要加入受版权限制的原文堆料、私有研究包、未公开客户材料或无依据结论。来源引用应保持描述性和简短。综合写作流程必须保留冲突、置信度和证据缺口。
