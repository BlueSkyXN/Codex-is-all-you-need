主应用内嵌

现有 SaaS 部署单元

业务数据库租户向量索引队列 / 作业

部署、鉴权和调用路径简单；索引任务、模型调用和主业务共享扩缩容与故障边界。

  • 适合团队较小、规模和租户模式稳定
  • 主应用发布会同时影响 RAG 能力

独立 RAG 服务

现有 SaaS→内部服务接口→RAG 服务
解析 / 索引检索 / 重排推理编排

可独立扩容、复用和隔离故障；增加服务接口、版本兼容和跨团队运维成本。

  • 适合多个产品复用或索引负载独立增长
  • 需要定义鉴权、幂等、超时和降级契约
